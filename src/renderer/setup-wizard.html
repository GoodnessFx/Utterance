Welcome to Utterance

The sermon companion that displays scripture as it is spoken

Utterance listens to your sermon and automatically displays Bible verses on screen as they are quoted. No need for a verse caller — just preach.

Choose Your Language

Select the language for the Utterance interface

This changes the app language. You can also set the Bible translation and transcription language separately in Settings later.

Select Display

Choose where scripture verses will be projected

If you have an external monitor or projector connected, select it here. Verses will appear fullscreen on that display.

🖥️
Primary Display

📽️
External Display
Projector or second monitor

You can change this later in Settings. The projection window can also be moved between monitors manually.

Lock Settings

Set a PIN to prevent accidental changes during service

A PIN locks the settings panel so the app cannot be accidentally modified during a service. You can skip this and set one later.

Keep this PIN private. It cannot be recovered if lost.
